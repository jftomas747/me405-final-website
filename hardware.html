Hardware

Base platform, added sensors, and mechanical layout

Base Platform

Our robot was built on the standard Romi platform used throughout the course. The core hardware stack included the Romi chassis with motors, encoders, wheels, and casters, along with a Nucleo L476RG, a modified Shoe of Brian, and a BNO055 IMU breakout board.

Added Sensors and Components

To support autonomous track navigation, we added a QTRX-MD-07A 7-channel analog reflectance sensor array and left/right bumper switch assemblies. These sensors allowed the robot to follow the printed track while also detecting wall contact during navigation.

Mechanical Layout

The line sensor was mounted near the front underside of the robot using stand-offs so it could read the track close to the ground. The bumper switches were mounted at the front so they would contact the wall before the line sensor or front hardware made impact. The IMU was incorporated into the robot stack to support heading estimation during turns.
