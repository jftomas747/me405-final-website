ME405 Final Project - MECH13

Romi obstacle course robot

Project Overview

This project combines motors, encoders, line sensors, IMU state estimation, and bump sensors into a Romi robot that can complete an obstacle course.

Team Members

Kaden Guevarra, John Tomas

Hardware and Design

We used a Romi chassis, motors, encoders, a BNO055 IMU, a QTR reflectance sensor array, and left/right bump sensors.

Results

Our robot completed the course successfully and demonstrated line following, turning, and obstacle interaction.
