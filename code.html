Code / Software Architecture

Our software was organized as a multitask robot control system running on the Nucleo. The user interface handled runtime configuration of gains and setpoints, the motor tasks performed closed-loop PI velocity control using encoder feedback, the line sensor task handled steering corrections, and the IMU/state-estimation tasks provided smoother heading and motion information for higher-level navigation.
